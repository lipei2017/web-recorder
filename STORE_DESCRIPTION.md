# Chrome 应用商店发布信息

## 简短描述（132 字符以内）
专业的网页请求录制与回放工具，支持 HTTP/WebSocket 请求捕获、数据导出和智能回放功能。

## 详细描述

WebRecorder 是一款强大的网页请求录制与回放工具，专为开发者和测试人员设计。

### 主要功能

🎯 **智能录制**
- 自动捕获 HTTP/HTTPS 请求（XMLHttpRequest 和 Fetch API）
- 完整记录 WebSocket 消息（发送和接收）
- 支持域名配置，自动录制指定网站
- 实时显示录制状态和请求数量

🔄 **精准回放**
- 智能匹配录制数据，支持不同环境回放
- WebSocket 消息循环播放
- 支持路径模式匹配（动态 ID 自动替换）
- 跨域名回放支持

📊 **数据管理**
- 会话管理：查看、重命名、删除录制会话
- 数据导出：支持 JSON 格式导出
- 高级过滤：按 URL、请求类型、状态码过滤
- URL 路径模糊匹配（支持通配符）

⚙️ **灵活配置**
- 域名白名单管理
- 支持通配符域名匹配
- 录制/回放模式切换
- 自动录制/回放配置

### 使用场景

- **前端开发**：录制 API 请求，离线开发调试
- **接口测试**：录制生产环境请求，在测试环境回放
- **Bug 复现**：录制问题场景，便于问题定位
- **演示展示**：录制完整交互流程，离线演示

### 技术特点

- 零配置启动，开箱即用
- 轻量级设计，不影响页面性能
- 完全本地存储，保护数据隐私
- 支持 Manifest V3，符合最新标准

### 隐私保护

- 所有数据仅存储在本地浏览器
- 不收集任何用户信息
- 不上传数据到任何服务器
- 完全开源，代码透明

### 快速开始

1. 安装扩展
2. 点击扩展图标打开控制面板
3. 点击"开始录制"按钮
4. 浏览网页，所有请求将被自动捕获
5. 点击"停止录制"保存会话
6. 在"记录管理"中查看和管理录制数据
