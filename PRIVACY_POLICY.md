# WebRecorder 隐私政策

最后更新日期：2025年12月

## 数据收集

WebRecorder 不会收集、存储或传输任何用户个人信息到外部服务器。

## 本地数据存储

- 所有录制的网络请求数据仅存储在您的浏览器本地存储中
- 使用 Chrome 的 IndexedDB 和 localStorage API 进行本地存储
- 数据不会上传到任何服务器或第三方服务

## 权限说明

WebRecorder 需要以下权限：

- **activeTab**: 访问当前活动标签页，用于注入录制脚本
- **storage**: 本地存储录制会话和配置
- **tabs**: 管理标签页状态
- **scripting**: 注入内容脚本以捕获网络请求
- **downloads**: 导出录制数据到本地文件
- **unlimitedStorage**: 存储大量录制数据
- **host_permissions (<all_urls>)**: 在所有网站上工作，捕获网络请求

## 数据安全

- 所有数据仅存储在您的本地浏览器中
- 您可以随时删除录制的会话数据
- 卸载扩展将自动清除所有存储的数据

## 数据导出

- 您可以将录制的数据导出为 JSON 文件
- 导出的文件完全由您控制，不会发送到任何服务器

## 联系方式

## 政策更新

我们可能会不时更新此隐私政策。任何更改将在此页面上发布。
